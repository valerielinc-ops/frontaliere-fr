  • Responsable de toutes les questions pertinentes sur l'automatisation des tests

  • Utilisation d'outils de test existants (à base de Java) pour automatiser et piloter des alternatives pour trouver des solutions techniques innovantes
  • Code de document, cas de test, résultats de test et concepts Formation complète dans le domaine de l'informatique ou des entrants latéraux techniquement qualifiés avec une très bonne connaissance Java
  • connaissance et expérience dans le développement et l'essai agiles de logiciels
  • Passion pour de bonnes solutions logicielles
