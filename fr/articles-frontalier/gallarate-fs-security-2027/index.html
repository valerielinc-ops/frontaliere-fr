Questions fréquentes
Quand le nouveau poste de Fs Security à Gallarate ouvrira-t-il?
Le nouveau poste de Fs Security à Gallarate ouvrira au cours de l'année 2027. L'objectif est d'anticiper les délais autant que possible, mais la date exacte n'a pas encore été spécifiée.
Combien d'opérateurs seront présents au poste de Gallarate?
Le poste de Gallarate sera initialement composé de 12 à 15 opérateurs, avec l'objectif d'assurer une couverture 24 heures sur 24, 365 jours par an.
Quelles sont les activités prévues pour les opérateurs de Fs Security?
Les opérateurs de Fs Security effectueront des contrôles des titres de transport, sanctionneront les passagers irréguliers et fourniront un soutien dans la gestion de la sécurité ferroviaire. Ils seront également équipés de bodycams pour les situations à risque.
