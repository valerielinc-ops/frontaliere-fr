Médias suisses adoptent un code éthique sur l'IA (guide frontalier)

Di Valerie Linc · Publié le 7 mai 2026

Un nouveau code de conduite pour l'utilisation responsable de l'intelligence artificielle dans les médias suisses, inspiré du Conseil de l'Europe, vise à renforcer la confiance du public.

Contesto

En bref - Les médias suisses adoptent un code de conduite sur l'IA - Le code s'inspire d'une convention du Conseil de l'Europe - Objectif : renforcer la confiance du public ## Faits clés - Quoi : Code de conduite sur l'IA pour les médias suisses - Quand : Présenté le 7 mai 2026 - Où : En marge du Swiss Media Forum à Lucerne - Qui : Association des éditeurs suisses-allemands (VSM), Médias Suisses, Stampa Svizzera, radios et télévisions privées, SSR, Keystone-ATS - Montant : Non spécifié Le secteur des médias en Suisse a introduit un code de conduite pour l'utilisation responsable de l'intelligence artificielle (IA). Cet outil, inspiré d'une convention du Conseil de l'Europe, vise à renforcer la confiance du public envers cette technologie. 'La confiance est le bien le plus précieux pour les médias', déclare Andrea Masüger, président de l'association des éditeurs suisses-allemands Schweizer Medien (VSM). Le texte, présenté aujourd'hui en marge du Swiss Media Forum à Lucerne, est également soutenu par les éditeurs de la Suisse romande (Médias Suisses), de la Suisse italienne (Stampa Svizzera), par les organisations des radios et télévisions privées, par la SSR et par Keystone-ATS. Le Conseil de la presse, qui s'occupe également des questions déontologiques liées à l'IA, accueille favorablement cette initiative. ### Implications pour les frontaliers Alors que certaines études mettent en évidence la méfiance du public envers les contenus générés avec l'aide de l'IA, le code de conduite doit agir comme un outil d'autorégulation. Il fait référence à la convention-cadre du Conseil de l'Europe sur l'intelligence artificielle et les droits de l'homme, signée fin mars 2025 à Strasbourg par le conseiller fédéral Albert Rösti. Analyse pratique : impacts pour les frontaliers Le nouveau code de conduite sur l'IA dans les médias suisses introduit diverses mesures qui peuvent influencer les frontaliers vivant en Italie et travaillant en Suisse. Voici quelques implications pratiques : ### Formation et transparence Les médias suisses sont tenus de former leur personnel à l'utilisation de l'IA, garantissant que les contenus soient produits de manière responsable. Cela peut influencer la qualité des informations disponibles pour les frontaliers, qui pourront compter sur des nouvelles plus précises et transparentes. De plus, les médias doivent informer le public de manière transparente sur leur utilisation de l'IA, par exemple via une page dédiée sur leur site web. Cela peut aider les frontaliers à mieux comprendre comment les contenus sont générés et à faire des choix éclairés. ### Protection des données Le code de conduite impose aux médias d'adopter des mesures pour protéger les données personnelles. Cela est particulièrement pertinent pour les frontaliers, qui doivent souvent gérer des informations sensibles relatives à leur statut professionnel et fiscal. La protection des données est fondamentale pour garantir que les informations personnelles des frontaliers soient traitées de manière sûre et confidentielle. ### Étiquetage des contenus Le code de conduite impose des obligations d'étiquetage contraignantes pour les contenus générés avec l'IA. Cela signifie que les médias suisses doivent clairement indiquer quand un contenu a été produit avec l'aide de l'IA. Cela peut aider les frontaliers à distinguer entre les contenus générés par des humains et les contenus générés par des systèmes d'IA, leur permettant de faire des choix éclairés. ### Mécanisme de signalement et de contrôle Le code de con...
